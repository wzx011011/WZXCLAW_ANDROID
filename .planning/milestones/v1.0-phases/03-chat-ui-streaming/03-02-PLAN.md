The file structure (single StatefulWidget `HomePage` with `_HomePageState`):

**State fields:**
- `_inputController` (TextEditingController)
- `_scrollController` (ScrollController)
- `_revealedTimestamps` (Set<int> -- tracks which message indexes have timestamp revealed)
- `_streamingSub` and `_messagesSub` (StreamSubscription for ChatStore streams)
- `_isStreaming` (bool, from ChatStore.streamingStream)
- `_displayMessages` (List<ChatMessage>, from ChatStore.messagesStream)

**initState:**
1. Call `_autoConnect()` (preserve existing auto-connect logic from Phase 1 -- same SharedPreferences URL construction with role=mobile and token).
2. Call `ChatStore.instance.loadHistory()`.
3. Subscribe to `ChatStore.instance.messagesStream` -- on each emission, update `_displayMessages` and call `_scrollToBottom()` if streaming.
4. Subscribe to `ChatStore.instance.streamingStream` -- update `_isStreaming`, which controls send vs stop button.

**dispose:** Cancel subscriptions, dispose controllers.

**build method -- Scaffold layout:**
