## Interaction Contracts

### Opening the Project Drawer
- Trigger: tap menu icon (hamburger) in AppBar leading position
- Drawer slides in from left (standard Material Drawer behavior)
- On first open: auto-send `/projects` command if project list is empty
- Subsequent opens: show cached project list, with a "refresh" icon button to re-fetch

### Project List Display
- List of project names, each as a `ProjectListTile`
- Left side: status dot (8px circle, green = AI running, grey = idle)
- Center: project name text
- Right side: if this is the current project, show a check icon (`Icons.check_circle`, accent color)
- Active project row has `#6366F1` at 12% opacity background tint

### Switching Project
- Tap any project in the list
- Send `/switch <project-name>` via `ChatStore.sendMessage()`
- Show brief confirmation in drawer: update active highlight immediately
- Close drawer
- Desktop response appears as assistant message in chat (existing behavior)
- If switch fails (desktop returns error): show error state, revert highlight

### Pull-to-Refresh
- Drawer project list supports pull-to-refresh (Material `RefreshIndicator`)
- On refresh: re-send `/projects` command

### Connection Dependency
- If disconnected when drawer opens: show "未连接 -- 无法获取项目" overlay on the project list area (similar to empty state)
- Re-fetch automatically when connection is restored

## Layout Specification

### Drawer Dimensions
- Width: `304px` (standard Material Navigation Drawer width for small screens)
- Background: `#1A1A2E` (dominant)

### Drawer Header
- Height: `120px`
- Background: `#16213E` (secondary)
- Bottom border: 1px accent (`#6366F1`) line, 3px height
- Content: "项目" title, 20px, white, left-aligned, padded 24px horizontal / 16px top
- Below title: current project name in 14px, `Colors.white70`

### Project List Item
- Height: `56px` minimum (standard Material list tile)
- Horizontal padding: `16px` left, `16px` right
- Layout: [8px status dot] [12px gap -- legacy exception] [project name] [flexible space] [check icon if active]
- Project name: 15px, `Colors.white` (or `Colors.white70` if not active)
- Active item background: `#6366F1` at 12% opacity
- Tap ripple: accent color at 12% opacity (Material default)

### Drawer Footer
- Padding: `12px` horizontal (legacy exception)
- Connection status label: 13px, status-colored text (reuse `WsConnectionState.label` and `_stateColor` pattern)
- Subtle top border: `Colors.white12`, 0.5px
